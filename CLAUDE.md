# API Astrológica - FastAPI & PySwissEph

## Comandos de Configuración
* Crear entorno virtual: `python -m venv venv`
* Activar entorno: `source venv/bin/activate` (Linux/Mac) o `venv\Scripts\activate` (Windows)
* Instalar dependencias: `pip install -r requirements.txt`
* Iniciar servidor: `uvicorn main:app --reload`

## Estilo de Código
* Usar tipado estricto de Python (`typing`).
* Validar datos de entrada con Pydantic v2.
* Formatear respuestas JSON con nombres de variables claros en español.
